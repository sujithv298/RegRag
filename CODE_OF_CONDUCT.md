# Code of Conduct

This project follows the [Contributor Covenant, v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be kind, assume good faith, focus on the work. Harassment, personal attacks, or discriminatory behavior are not tolerated and will result in removal from the project.

Report concerns to the maintainer listed in `pyproject.toml`.
